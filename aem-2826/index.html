3. VPN programları

VPN nedir?
Virtual Private Network (Sanal Özel Ağ) teriminin kısaltılmışı olan VPN, basit bir anlatımla internet üzerinde kurulu bilgisayarlar grubundan oluşur. Bu özel bilgisayar ağı, örneğin yerel bir ağa fiziksel erişimi bulunmayan, dışarıdaki bir kişi tarafından ağ kaynaklarına erişmekte kullanılabilir. VPN’ler, güvenilmeyen ağlara bağlanırken bağlantıyı şifrelemek ve güvenli hale getirmek için de kullanılır. Bunun yanında VPN’ler, farklı konumlarda bulunabildiğinden, sizi ülkelerin kısıtlamalarından (yasaklı web siteleri ve IP’ler gibi) da kurtarırlar.
VPN’lerle bağlantınız şifreli olduğundan, gönderdiğiniz ve aldığınız tüm veriler şifrelendiğinden, meraklı gözler ne yaptığınızı göremezler.
VPN’i sadece engelli sitelere erişim sağlayabilmeniz için anlatıyoruz. Bazı VPN şirketleri mahkeme kararlarına uyup istendiği takdirde bilgilerinizi otoritelere verebilirler, bundan dolayı internette ne yaptığınıza dikkat edin.

4. Proxy üzerinden bağlantı kurmak

Proxy nedir? Vekil sunucu veya Yetkili sunucu, internete erişim sırasında kullanılan bir ara sunucudur. Kullanıcılar, özellikle hükümet tarafından yasaklanan (teknik tabiriyle “erişimi engellenen”) internet sitelerine bağlanmak için bu yönteme başvururlar.

4.1 Windows Phone

Bu bağlantı tarzı bildiğimiz kadarıyle yanlız Windows 8.1’re güncellenen Windows Phone’larda çalışır. Hesap eklemelerde bazen bağlanmayabilir, yeniden deneyin.

4.1.1 WiFi bağlantısı (Proxy) // Video: TIKLA

“Ayarlar” => “Wi-Fi” => bağlı olduğunuz Wi-Fi’ya tıklayın => “Proxy”yi açık konuma getirin => 5.3‘de olan listeden bir IP/Proxy adresi ve ona bağlı olan Port numarasını seçin, kayıt et ve sansürsüz giriş yapın

4.1.2 Veri bağlantısı (Proxy) // Video: TIKLA

“Ayarlar” => “Şebeke” => Veri bağlantısını kapatın => “Ayarlar”a geri dönün => “Wi-Fi” => bağlı olduğunuz Wi-Fi’ya tıklayın => “Proxy”yi açık konuma getirin => 5.3‘de olan listeden bir IP/Proxy adresi ve ona bağlı olan Port numarasını seçin ve kayıt et => “Ayarlar”a dönün => “E-posta + Hesablar”=> “Hesap ekle”=> sosyal medya hesabını ekleyin ve giriş yapın => “Ayarlar”a dönün => “Wi-Fi” => bağlantıyı kesin => “Ayarlar”a dönün => “Şebeke” => Veri bağlantısını açın => ana ekrana geri dönün => “Kişiler” => “Yeni” bölümüne girin => burada “twitter timeline” gösterilir, malesef yanlız ReTweet ve tweetlere cevap verebilirsin.

6.1 Windows 7.x ve 8.x işletim sisteminde DNS değiştirme:

Ağ ve network Merkezinde sağ alt köşesinde olan WiFi sembolüne basın => Bağlantılar: <senin bağlı olduğun WiFi network isimi> (tıkla) => Özellikler => Internet Protokol sürüm 4 (TCP/IPv4) => Özellikler => Aşadaki DNS sunucu adreslerini kullan: (işaretle)

”Terceh edilen DNS sunucusu:”  “208 . 67 . 222 . 222” ekle
“Diger DNS sunucusu:“ ”208 . 67 . 222 . 220” ekle

“Diğer DNS sunucusu:” boş bırakabilirsiniz.

IPv6 adresleri kullanmak için ”Internet Protokol sürüm 4 (TCP/IPv4)” yerine ”Internet Protokol sürüm 4 (TCP/IPv6)”“ seç ve aynı şekilde devam et.
Modemi veya bağlantıyı kesmek ve yine kurmak bağlantı problemini çözebilir. DNS bağlantısını Windows 8.x altında değiştirmek için sağ alt köşede olan WiFi sembolüne mousesa sağ padla tıkla ve aynı Windows 7.x olduğu gibi devam et. (Video olarak: TIKLA)

6.2 Apple Mac OSX işletim sisteminde DNS değiştirme:

Bilgisayarınızın menüsünden ”Sistem Özellikleri” => Ağ kısmına” tıklayın => Gelişmiş” butonuna tıklayın => 2. Sekmeden ”DNS” kısmına basın => „+“ butona tıklayarak verilen DNS adreslerini ekleyin: ”208 . 67 . 222 . 222“ ve ”208 . 67 . 222 . 220“ => Ayarları kaydedip, modemi kapatıp aç.

6.3 Android işletim sisteminde DNS değiştirme:

Ayarlar => Kablosuz (baglanti) => bağlı olduğunuz ağın üzerine parmağınızı basılı tutun => ”Ağı değiştir“ seç => “gelişmiş seçenekleri göster” seçeneğini işaretleyin => IP Ayarları, DHCP olarak seçilidir. Bunu Statik yapıp ve aşağıdaki DNS 1 ve DNS 2 değerlerini sırasıyla OPENDNS’in adresleri olan ”208.67.222.222” ve ”208.67.222.220” değerleri ile değiştirip => İnternet bağlantınızı kapatıp ve tekrar aç.

6.4 iOS işletim sisteminde DNS değiştirme:

Ayarlar => WiFi (Aglari) => bağlı olduğunuz ağı seçip ”OK” seçeneğine tıklayın => iOS işletim sisteminde tek bir DNS adresini desteklediği için bir tane yazacaksınız => Örnek olarak Google DNS adresini kullanırsanız ”208.67.222.222” numarasını yazmak yeterli olacaktır => İnternet bağlantınızı kapatıp ve tekrar aç.
